Selecteur

Les selecteur dans le css permettent de designer une balise ou une série de balises précises dans le fichier Html

[élément]:nth-child()

nth-child permet de selectionner une balise enfant depuis sa balise parent
[Exemple] :
li:nth-child(even)
(le selecteur choisi les balises paires de la balise parent de li)

  • Child 1
  • Child 2
  • Child 3
  • Child 4
  • Child 5

[élément]:nth-of-type()

nth-of-type permet de selectionner une balise parmit la liste de balise du même type
[Exemple] :
tr:nth-of-type(odd)
(le selecteur choisi les balises impaires de toutes les balises tr)

Type tr 1
Type tr 2
Type tr 3
Type tr 4
Type tr 5

id & class

les atributs de balises id & class sont utilisé pour désigner spécifiquement une ou plusieurs balises entre le html et le css
[Exemple] :

HTML :

<img class="logo" src="Img/black-mesa-logo.png">

CSS :

.logo {width: 15%}

(le selecteur choisi les balises impaires de toutes les balises tr)
